NPM

npm (node Package Manager) est un gestionnaire de paquets pour l'écosystème Node.js. Il est utilisé pour installer, mettre à jours et gérer les dépendances des projets Nodes.js. NPM permet aux développeurs de partager et de reutiliser des bibliothèques de code (appelées "paquets" ou "modules") développés par d'autres, ainsi que de publier leur propres paquets pour être utilisés par la communauté.

Le TypeScript

Le TypeScript est un language de programmation open source développé par Microsoft. Il est conçu comme un sur-ensemble typé de JavaScript; ce qui signifie que tous code JavaScript valide est également du code TypeScript valide. Cependant, TypeScript offre des fonctionnalités supplémentaires qui améliorent le développement et la maintenance des applications JavaScript à grande échelle..

Les principales fonctionnalités du TypeScript

  1. Typage statique : TypeScript permet de définir des types pour les variables, les paramètres de fonctions, les valeurs de retour et d'autres éléments du code. Cela permet de détecter les erreurs de typage lors de la phase de developpement plutôt qu'à l'execution, ce qui peut améliorer la fiabilité du code.

Les types de variable en JavaScript

  • Number: les nombres, qu'ils soient entiers ou à virgule
  • string :represente les chaîne de caractères
  • boolean: represente les valeurs booléennes, c-a-d true ou false
  • array: represente i, tableau contenant des éléments du même type
  • object:represente un tableau contenant des éléments du même type
  • null: répresente la valeur "null
  • undefined: represente la valeur "undefined"
  • any: represente un type non spécifique ou dynamique.
  • unknown: represente un élément dont on ne connait pas le type
  • enum: represente un ensemble de valeur nommées
  • tuple: represente un tableau avec un nombre fixe d'éléments du types différents
